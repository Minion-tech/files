# LexAlly AI Architecture Quiz
### by Nabeela Khan

An interactive Streamlit quiz to test knowledge of AI Agent patterns and RAG techniques.

---

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## Deploy to Streamlit Community Cloud (free)

1. Push this folder to a **GitHub repo** (public or private)
2. Go to → [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub
4. Click **"New app"**
5. Select your repo, branch (`main`), and set **Main file path** to `app.py`
6. Click **Deploy** — you'll get a public URL like:
   `https://your-app-name.streamlit.app`
7. Share that link on LinkedIn!
